Задание 3: Викторина по орфографии русского языка

Это задание демонстрирует создание интерактивной викторины с использованием JavaScript. Викторина содержит вопросы по орфографии русского языка с несколькими вариантами ответов. После выбора ответа пользователь получает обратную связь о правильности своего ответа.

В викторине реализована система случайного перемешивания вопросов и ответов при каждом запуске, что делает прохождение более интересным. После ответа на вопрос показывается анимация, которая визуально демонстрирует правильность или неправильность выбора.

После завершения викторины пользователь может просмотреть статистику правильных ответов и пройти викторину заново. Все вопросы можно просмотреть повторно, кликнув на них, чтобы увидеть правильные ответы и пояснения.

Это задание демонстрирует работу с DOM-элементами, обработку событий, создание динамического контента, анимации и управление состоянием приложения с помощью JavaScript. Также используются современные CSS-техники для создания красивых переходов и анимаций.

Технические детали

Для реализации викторины использованы следующие технологии:

  • JavaScript - для логики викторины, обработки ответов и управления состоянием
  • DOM API - для динамического создания и изменения элементов страницы
  • Event Listeners - для обработки кликов и взаимодействия пользователя
  • CSS Animations - для плавных переходов и визуальных эффектов
  • CSS Transitions - для анимации изменения свойств элементов
  • Flexbox - для адаптивной верстки и расположения элементов
  • Array Methods - для работы с данными (map, forEach, find, sort)

Викторина использует алгоритм Fisher-Yates для перемешивания вопросов и ответов, что обеспечивает случайный порядок при каждом запуске. Все вопросы хранятся в массиве объектов, что позволяет легко добавлять новые вопросы или изменять существующие.
