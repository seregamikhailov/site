Задание 1: Аквариум

Это задание демонстрирует создание графической композиции "Аквариум" исключительно средствами CSS. В работе используются различные техники CSS для создания сложных фигур: рыбы, водоросли, песок на дне аквариума.

Все элементы созданы с помощью CSS свойств: градиенты, border-radius, clip-path, псевдоэлементы ::before и ::after. Это задание показывает, насколько мощным инструментом является CSS для создания графики без использования изображений или JavaScript.

В композиции представлены три рыбы разных размеров и цветов, водоросли с интересной формой, созданной с помощью clip-path, и дно аквариума с песком, на котором размещены декоративные элементы. Каждый элемент тщательно проработан и использует современные возможности CSS.

Для создания рыбок использованы градиенты, которые создают объем и реалистичность. Хвосты рыб созданы с помощью clip-path, что позволяет создавать сложные формы. Глаза рыб - это простые круги, а узоры на телах созданы с помощью псевдоэлементов и border-radius.

Технические детали

Для создания композиции использованы следующие CSS техники:

  • linear-gradient - для создания градиентных фонов и объемных эффектов
  • radial-gradient - для создания круглых градиентов
  • border-radius - для скругления углов и создания круглых форм
  • clip-path - для создания сложных форм (хвосты рыб, водоросли)
  • ::before и ::after - псевдоэлементы для добавления дополнительных деталей
  • transform - для поворота, масштабирования и позиционирования элементов
  • position: absolute - для точного позиционирования элементов

Аквариум создан с помощью кнопки "Показать/Скрыть", которая использует JavaScript для переключения класса. Это демонстрирует взаимодействие CSS и JavaScript для создания интерактивных элементов.
