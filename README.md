# Портфолио веб-разработчика

Учебный проект по веб-разработке, демонстрирующий различные техники верстки и программирования.

## Структура проекта

```
site/
├── index.html              # Главная страница
├── pages/                  # Страницы заданий
│   ├── assignment1.html   # Задание 1: Аквариум
│   ├── assignment2.html   # Задание 2: Интерактивная анимация
│   └── assignment3.html   # Задание 3: Викторина
├── assets/
│   ├── css/               # Стили
│   │   ├── main.css       # Основные стили сайта
│   │   ├── assignment1.css # Стили для задания 1
│   │   ├── assignment2.css # Стили для задания 2
│   │   └── assignment3.css # Стили для задания 3
│   ├── js/                # JavaScript файлы
│   │   ├── main.js        # Основной скрипт (навигация, мобильное меню)
│   │   └── quiz.js        # Скрипт викторины
│   └── images/            # Изображения
│       ├── main/          # Изображения для главной страницы
│       └── assignments/  # Изображения для заданий
└── README.md              # Этот файл
```

## Особенности

- ✅ Адаптивный дизайн
- ✅ Семантическая HTML5 разметка
- ✅ CSS анимации и переходы
- ✅ Интерактивные элементы на JavaScript
- ✅ Организованная структура файлов
- ✅ Чистый и читаемый код

## Задания

1. **Аквариум** - Графическая композиция, созданная исключительно средствами CSS
2. **Интерактивная анимация** - Сцена с CSS анимациями и переходами
3. **Викторина** - Интерактивная викторина по орфографии с JavaScript

## Технологии

- HTML5
- CSS3 (Flexbox, Grid, Animations)
- JavaScript (Vanilla JS)
- Адаптивная верстка
